# Gold-Currency

This Application is for Monitoring  Gold and Currencies prices. 


# IOS Application Development

## Outlines 

- #### Swift 5
- #### Xcode
- #### MVVM Design Pattern
  - MVVM stands for Model-View-ViewModel. MVVM design pattern introduces a fourth component, the view model. 
  The view model is responsible for managing the model and funneling the model's data to the view via the controller. This is what that looks like.

  ![figure-model-view-viewmodel-3](https://user-images.githubusercontent.com/117483832/203425932-ed1383f2-7c29-4417-847e-d7cd430a2184.jpg)
  
- #### User Settings
In This Application, we provide user setting to let our application be more user friendly.
  - ###### Dark Mode Featue
  - ###### Setting The Default Language Featue
  - ###### Setting The Default Metal Type Featue
  - ###### Setting The Default Currency Featue

- #### Getting Diferent Metal Prices
  - In this Apllication You Can get the prices of Gold, Silver, Platinum, and Palladium.
  - In this Apllication You Can get the prices of Different Metal with around 25 National Currencies.

- #### Gold API
  - You can Learn More About This API Through This Link: https://www.goldapi.io/dashboard

- #### Exchange Currency API
  - You can Learn More About This API Through This Link: https://anyapi.io/app
- #### Xcode
